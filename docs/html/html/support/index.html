Support¶

Getting help¶

The easiest wayto get help on QUINOPT’s functions is to use the help command in MATLAB, and look at the source code. If you still need help, you can contact us. If you find a bug, or have an issue, see the next section.

Bug reports and support¶

Please report any issues via the Github issue tracker, or contact us. All types of issues are welcome, including bug reports, documentation typos, feature requests and so on.
