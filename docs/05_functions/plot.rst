``@legpoly/plot()``
================================================

Short description

---------------------

:Syntax: ``plot(X,P)``
:Description: plots the Legendre polynomial ``P`` at the points specified by the vector ``X``. All points in ``X`` must be within the domain of definition :math:`[a,b]` of the Legendre polynomial, as returned by calling ``getDomain(P)``.

---------------------

:Syntax: ``PLOT(X,P,LINESPEC)``
:Description: applies the formatting specified by ``LINESPEC``, as in the usual ``plot()`` command in MATLAB.
