================================================================================
                              QUNOPT Examples
================================================================================

This folder contains demo files for QUINOPT, named "exampleXX.m". A description 
of each example, and a step-by-step guide through the code and the functions used, 
can be found in the online documentation at:

quinopt.readthedocs.io/

The folder "./paper/" contains matlab files to reproduce the results presented
in the following paper:

  G. Fantuzzi, A. Wynn, P. Goulart, A. Papachristodoulou, "Optimization with 
  affine homogeneous quadratic integral inequality constraints", arXiv:1607.04210
  [math.OC], 2017 (to appear in IEEE Transactions on Automatic Control).
